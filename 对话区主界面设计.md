## 二、Mini Nav（52px 左侧导航）

### 交互
- 5 个图标按钮，竖排
- 点击切换激活态：白字白图标 + 微亮背景
- 非激活态：灰字灰图标 + 透明背景
- 背景：从上到下黑色渐变透明

### 导航项
| 图标 | 标签 | 功能 |
|------|------|------|
| 💬 | 通信 | 切换到聊天主视图 |
| 👥 | 識別 | 切换到角色/Agent 管理 |
| 📋 | 記録 | 切换到任务/会话记录 |
| 💰 | 資源 | 切换到费用/用量仪表盘 |
| ⚙ | 設定 | 打开全屏设置页面 |

### 状态
- 每个 Nav item 独立激活态
- 点击后更新右侧对应内容区，或触发面板切换

## 三、Sidebar（320px 侧边栏）

### 3.1 Section 1：当前会话

- **头像**：36px 圆角方形，显示角色首字母
- **名称 + 角色**：Claude Opus 4.6 / 技術顧問 · 戦術支援
- **时间**：JetBrains Mono 等宽数字
- **轮次**：通信 #N 格式

交互：
- 点击区域 → 切换到该 Agent 的主会话
- 头像支持自定义图片

### 3.2 Section 2：搜索

- placeholder："搜索通信记录..."
- 输入即搜，下方实时过滤会话列表
- 支持 ID 搜索和标题搜索

### 3.3 Section 3：角色列表

- 多 Agent 树形结构
- 每个 Agent 行：小头像(32px 圆角方形) + 名称 + 角色描述
- 展开/折叠箭头 `▸` / `▾`
- 子会话列表：
  - 激活会话：左侧细竖线标记
  - 点击切换会话
  - 每个会话显示标题 + 轮数
  - hover 显示删除按钮 `×`
  - 删除需二次确认

### 3.4 Section 4：作戦記録（工具调用）

- 标题行：标签 + 暂停按钮
- 暂停按钮：中断当前 AI 输出
- 工具列表：图标 + 工具名 + 参数摘要 + 状态
- 状态：执行中(动画) / 成功(✓ + 耗时) / 失败(✗)
- 点击展开详情 → 显示输入 JSON + 输出文本

### 3.5 Section 5：資源監視（费用仪表盘）

- Token 卡片：输入/输出/总计 + 进度条 + 压缩按钮
- 费用卡片：本次/本月/总计费用
- 压缩按钮：触发上下文压缩，小号紧凑样式
- 进度条颜色阈值：
  - < 60%：正常灰
  - 60-80%：警告黄
  - > 80%：危险红

## 四、Chat Panel（760px 聊天区）

### 4.1 标题栏

- 纯黑底 `#000000E6` + 白字
- 左侧：会话标题（如"澪号端末 · MIO-TERMINAL"）+ 宋体
- 右侧：PRTS 状态指示器（三点 + "PRTS 接続中..."）
  - 三点递减透明度表示在线状态

### 4.2 消息列表

消息渲染：
- **Markdown**：AI 回复中的 Markdown 语法实时渲染为 HTML（marked 库，breaks + gfm）
- **HTML**：AI 回复中若包含 HTML 片段，渲染为富文本
- **代码块**：深黑底 + 终端绿代码高亮 + JetBrains Mono
- **图片**：支持 `![alt](url)` 语法，直接渲染
- **纯文本**：无格式文本直接展示
- 已完成消息缓存渲染结果，避免重复解析闪烁

布局：
- 用户消息：右对齐，深色气泡 + 4px 圆角方形头像
- 澪号消息：左对齐，32px 头像 + 名称 + 气泡
- 同角色连续消息：仅第一条显示头像

气泡：
- 用户：深色 `#14141E80`，小圆角
- 澪号：<mark>待调整</mark>

思考块：
- 默认折叠，显示 `💭 思考过程 ▸`
- 点击展开 → 显示 `💭 思考过程 ▾` + 浅色背景内容区
- 字体 12px，灰色

消息底部：
- 小字 token 计数，极淡（`#FFFFFF2A`，10px）
- 格式：`241 tokens` / `1.2k tokens`

附件：
- 支持图片和文件附件
- 发送前显示附件徽章（文件名 + 类型图标）
- 可移除

### 4.3 输入区

- 附件按钮 + 输入框 + 发送按钮
- 输入框：暗色凹陷（`#040408`），4px 圆角，自适应高度(max 120px)
- 发送按钮：暗灰方角，Enter 发送，Shift+Enter 换行

工具快捷栏：
```
[MCP 工具] [代码片段] [图片] [文件]
```
- 4 个 chip 按钮，细边框
- 点击切换输入模式或打开选择器

拖放：
- 支持从文件树拖拽文件到聊天区
- 拖拽区域高亮

## 五、Editor Panel（340px 文件编辑）

### 5.1 标题栏

- 纯黑底 + 白字："資料閲覧"
- 右上角：收起按钮（"收起 ◀"）

### 5.2 Tab 栏

- 打开的文件 Tab，水平排列
- 激活 Tab：白色下划线 + 白字
- 非激活 Tab：灰字
- 每个 Tab：文件图标 + 文件名（JetBrains Mono）
- 支持关闭 Tab

### 5.3 工具栏

- 左侧：文件元信息（格式 · 行数）
- 右侧：预览切换按钮（**仅 .md / .html 显示**）

**预览按钮显示规则**：

| 文件类型 | 工具栏 |
|---------|--------|
| `.md` `.mdx` | `MARKDOWN · 18 LINES` \| `源码` \| `预览` |
| `.html` `.htm` | `HTML · 42 LINES` \| `源码` \| `预览` |
| `.tsx` `.ts` `.js` `.json` `.css` 等 | `TYPESCRIPT · 120 LINES`（无预览按钮） |
| 图片 (`.png` `.jpg` 等) | 直接渲染，无源码/预览切换 |

### 5.4 内容区

- 源码模式（所有文件）：等宽字体（JetBrains Mono），行号，终端绿 `#5B8C5A` 代码高亮
- Markdown 预览（仅 .md）：marked 渲染 HTML，innerHTML 注入
- HTML 预览（仅 .html）：iframe sandbox 渲染
- 图片：Base64 内嵌显示
- 代码块：深黑底 + 终端绿文字

## 七、全局状态

### 连接状态
- 在线：PRTS 狀態點呈現，狀態文字 "PRTS 接続中..."
- 离线：狀態點灰色，"接続断開"
- 流式中：狀態點綠色脈衝，"処理中..."

### 通知/Toast
- 中断操作："已中断，可继续对话"
- 压缩完成："已压缩，释放 X tokens"
- 错误：3 秒自动消失

### 快捷键
- `Enter`：发送消息
- `Shift+Enter`：换行
- `ESC`：关闭设置 / 取消操作
